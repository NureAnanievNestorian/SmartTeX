Підключення SmartTeX до AI-асистентів

SmartTeX підтримує інтеграцію через MCP (Model Context Protocol) — стандарт для підключення AI-асистентів до зовнішніх інструментів. Після підключення асистент зможе читати та редагувати ваші LaTeX-документи, запускати компіляцію і працювати з файлами проєкту.

Зміст

  1. Claude
  2. ChatGPT
  3. McpSuperAssistant
  4. Безпека
  5. Перевірка підключення

1. Claude

  1. Відкрийте Settings → Connectors у Claude.
  2. Натисніть Add custom connector, вкажіть довільну назву і вставте MCP URL SmartTeX.
  3. Claude відкриє вікно авторизації SmartTeX — увійдіть у свій акаунт і підтвердіть доступ.
  4. Після підключення інтеграція з'явиться у списку активних.

2. ChatGPT

Вимоги: платний тариф
Якщо у вашому тарифі немає можливості використання власних MCP-серверів, скористайтесь McpSuperAssistant.
  1. Відкрийте Settings → Apps → Advanced Settings і увімкніть Developer Mode.
  2. Натисніть Create App
  3. Вкажіть довільну назву, вставте MCP URL SmartTeX і пройдіть авторизацію у вікні підтвердження.
  4. Після підключення перейдіть до перевірки.

3. McpSuperAssistant

McpSuperAssistant — розширення для браузера, яке додає підтримку MCP до ChatGPT, Perplexity та інших веб-інтерфейсів. SmartTeX використовує кастомний форк із виправленнями сумісності.

Крок 1 — Встановлення розширення

Завантажте кастомний форк зі сторінки релізів: Source code (latest release)

Сторінка релізів: github.com/NureAnanievNestorian/MCP-SuperAssistant/releases
Оновлюю дані останнього релізу…

Розпакуйте архів у зручну папку, потім:

Chrome / Chromium / Edge / Brave
  1. Відкрийте chrome://extensions
  2. Увімкніть Developer mode у правому верхньому куті
  3. Натисніть Load unpacked і оберіть розпаковану папку
Firefox
  1. Відкрийте about:debugging#/runtime/this-firefox
  2. Натисніть Load Temporary Add-on і оберіть файл manifest.json з папки розширення

Крок 2 — Підключення SmartTeX

  1. Відкрийте ChatGPT або інший підтримуваний інтерфейс.
  2. Знайдіть кнопку MCP біля поля вводу і увімкніть перемикач — відкриється бічна панель.
  3. Вставте MCP URL SmartTeX у налаштуваннях сервера і натисніть Connect.
  4. Відкриється вікно авторизації SmartTeX — увійдіть і підтвердіть доступ.
  5. У бічній панелі має відображатись статус Connected.
{{ mcp_url }}

Крок 3 — Перший запит

Перш ніж давати завдання, вставте MCP-інструкційний промпт у розмову. Для цього натисніть кнопку MCP і оберіть Insert — це допоможе асистенту правильно використовувати інструменти SmartTeX.

Після цього перейдіть до перевірки підключення.

Режими виконання інструментів

Auto — інструмент виконується автоматично, результат одразу передається асистенту. Зручно для швидкої роботи.
Manual — виклик інструменту з'являється в бічній панелі для перегляду. Ви самі натискаєте Run, а потім Submit, щоб передати результат асистенту. Рекомендовано якщо плануєте редагування документів — так ви бачите що саме відбувається перед тим як зміни застосуються.

Безпека

  • Підключайте лише ті клієнти, яким довіряєте.
  • Після першого входу повторно авторизуватись не потрібно — клієнт оновлює доступ автоматично.
  • Якщо підключили чужий або помилковий клієнт — відкличте доступ у Settings → Integrations у SmartTeX і підключіть повторно.
  • Для публічних демо або тестування використовуйте окремий тестовий акаунт.

Перевірка підключення

Використовуючи SmartTeX, перевір чи маєш ти доступ до моїх проєктів і покажи їх список

Якщо підключення успішне — асистент поверне список ваших проєктів із SmartTeX.

Якщо щось пішло не так — переконайтесь що:

  • ви підтвердили доступ у вікні авторизації SmartTeX;
  • MCP URL введено правильно;
  • у вашому тарифі є підтримка MCP (актуально для ChatGPT).
